# Blinkyoo AI 👁️

**Uninterrupted focus. No history tracked. Localized intelligence.**

Blinkyoo is a premium, AI-driven productivity companion designed to keep you in the flow state. By leveraging real-time computer vision and native OS integration, Blinkyoo ensures you stay on task without compromising your privacy.

## ✨ Features

- **Native OS Screen Tracking**: Define an "Allowed List" of applications and browser tabs. Blinkyoo monitors your foreground window and alerts you the moment you stray off-target.
- **AI Attention Detection**: Powered by MediaPipe, the system detects if you are away from your desk or if you've picked up your phone, triggering instant reminders to return to focus.
- **Intelligent Session Evaluation**: At the end of every sprint, our integration with Groq AI analyzes your focus metrics (tab switches, away time, distractions) to provide personalized, actionable feedback.
- **Privacy First**: All camera processing happens locally in your browser. No video data ever leaves your machine. Blinkyoo operates statelessly—no history is saved, and no data is sold.
- **Glassmorphism Aesthetic**: A stunning, violet-themed interface designed for the modern desktop, featuring smooth animations and a non-intrusive HUD notification system.

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React, Framer Motion, Lucide React, Tailwind CSS (Custom Glassmorphism).
- **Backend**: FastAPI (Python), Ctypes (Windows API Integration), Groq Cloud API.
- **AI/ML**: MediaPipe (Face Landmesh & Object Detection).
